# Git
## Neues lokales Repository erstellen
```
git init
```

## Untracked files stagen
```
git add .
```

## Staged files commiten
```
git commit
```

## Commits angzeigen
```
git log
```
## Remote hinzufügen
```
git remote add origin <http_url>
```

## Commits pushen
```
git push origin master
```

## Commits vom Remote holen
```
git pull origin master
```
